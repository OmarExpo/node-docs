Deploying to Github for CI purpose

We need to first create a repository in Github including .gitgnore, readme.MD file & optionally licence MIT and then follow the steps below

  • git init (to intiliaze your project - assuming you are already in your project folder)
  • git add . (this will stage all your files excluding node_modules)
  • git commit -m "message"
  • git remote add origin 'repository url' (this is added your repository as remote-repo
  • git pull origin main (this will pull first before pushing as it is best practice)
  • git push (to push to the remote repo)
