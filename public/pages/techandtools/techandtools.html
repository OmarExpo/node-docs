Technologies and Tools

The technology stacks we have used in nodeJS class are:

  • IDE (i.e., VScode)
  • NodeJS (runtime environment for JavaScript)
  • N.P.M (Node Package Manager)
  • Express (framework)
  • nodemon (utility for server automatic restart)
  • Build tool (i.e., package.json)

Note: node.js is runtime environment for both framework and language but not either of them.

As a side note, in the build tool 'package.json', we have three things:

  • Dependencies
  • Build
  • Meta data
